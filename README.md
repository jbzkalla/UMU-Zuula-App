# UMU-Zuula: Lost and Found System 🛡️

**UMU-Zuula** is a premium, real-time Lost and Found management platform developed for university campuses. It streamlines the process of reporting lost items, claiming found belongings, and managing student-administrator communications.

---

## 🚀 Key Features

- **Dynamic FAQs**: Administrators can manage system questions in real-time.
- **Real-Time Messaging**: Two-way chat between students and administrators.
- **Secure Authentication**: Robust student and admin login using bcrypt encryption.
- **Administrative Suite**: Full CRUD (Create, Read, Update, Delete) control over items, claims, and users.
- **Live Analytics**: Detailed reports on item resolution and user feedback.
- **Professional Branding**: Tailored for the UMU university identity (#9B0F06, #134E8E).

## 🛠️ Installation & Setup

1.  **Database**: Import `database/lfis_db.sql` into your PHPMyAdmin.
2.  **Configuration**: Ensure `config.php` has the correct database credentials.
3.  **Permissions**: Make sure the `uploads/` directory has write permissions for item images.
4.  **PHP Version**: Recommended PHP 7.4 or 8.x with PDO and MySQLi enabled.
